# Simple Interest Calculator

This project contains a simple Bash script to calculate simple interest.

## Description

The calculator computes simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

The user enters:
- Principal amount
- Rate of interest
- Time period

The script then displays the calculated simple interest.
